# Garo Estate Website

## Description

Garo Estate is a comprehensive real estate platform that allows users to explore, filter,
and inquire about properties. The website features intuitive navigation and essential tools 
for property management, making it easy for users to browse listings, manage accounts, 
and communicate with property owners.

## Features
User Management:

1. Login & Sign-Up Pages: Secure user authentication and account creation.
   
2. User Profiles: Each user can upload a photo and maintain details such as name, title, email, phone, address, state, and personal info.

Properties Management:

1. Filter Options: Users can filter properties based on criteria like price, location, property type, bedrooms, and bathrooms.

2. Sort Options & Pagination: Sort properties by various factors and navigate through results with pagination.

3. Property Details: Each property page includes information such as price, location, area, type, bedrooms, bathrooms, city, description, amenities, and contact details for the owner.
   
Messages:

1. Users can send and receive inquiries about specific properties, with a dedicated messaging feature for each property owned.

APIs:

1. APIs available for user, city, favorite, inquiry, and property management, ensuring smooth data interaction across the platform.
